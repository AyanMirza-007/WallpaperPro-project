🖼️ WallpaperPro – A full-stack wallpaper sharing platform built with PHP, MySQL, HTML/CSS &amp; JS. Upload, browse by category, like, and download wallpapers. 
Application required: XAMPP


# WallpaperPro — Professional

Drop this folder into `xampp/htdocs/WallpaperPro`.

1.Launch xampp->Start Apache and Mysql click on admin.
2. Import `database.sql` into phpMyAdmin.
3. Edit `includes/config.php` with your DB credentials.
4. Ensure `uploads/` is writable.
5. Open `http://localhost/WallpaperPro/`.
